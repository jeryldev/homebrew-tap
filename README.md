# Homebrew Tap

Homebrew formulas for tools by [Jeryl Estopace](https://github.com/jeryldev).

## Installation

```zsh
brew tap jeryldev/tap
```

## Available Formulas

### dev-session-manager

Quick development session bootstrapping with tmux.

```zsh
brew install jeryldev/tap/dev-session-manager
```

Or after tapping:

```zsh
brew install dev-session-manager
```

**Repository:** [jeryldev/dev-session-manager](https://github.com/jeryldev/dev-session-manager)
